## 🔗 خريطة الربط بين الصفحات

```
لوحة التحكم (/)
├── المشتريات
│   ├── الموردين (/purchases/suppliers)
│   │   └→ فواتير الشراء
│   │   └→ أوامر الشراء
│   │   └→ المصروفات
│   ├── فواتير الشراء (/purchases/invoices)
│   │   └→ الموردين
│   │   └→ أوامر الشراء
│   │   └→ المخزون
│   ├── أوامر الشراء (/purchases/orders)
│   │   └→ الموردين
│   │   └→ فواتير الشراء
│   └── المصروفات (/purchases/expenses)
│       └→ الموردين
│       └→ فواتير الشراء
├── المبيعات
│   ├── العملاء (/sales/customers)
│   │   └→ فواتير البيع
│   │   └→ أوامر البيع
│   ├── فواتير البيع (/sales/invoices)
│   │   └→ العملاء
│   │   └→ أوامر البيع
│   │   └→ المخزون
│   └── أوامر البيع (/sales/orders)
│       └→ العملاء
│       └→ فواتير البيع
└── المخزون (/inventory)
    └→ فواتير الشراء
    └→ فواتير البيع
```

## ⚙️ كيفية الاستخدام

### 1. تشغيل النظام
```bash
npm run dev
```
افتح: http://localhost:3000

### 2. إضافة بيانات جديدة
البيانات التجريبية موجودة بالفعل! يمكنك:
- إضافة موردين جدد
- إضافة عملاء جدد
- إضافة منتجات جديدة
- إنشاء فواتير شراء (ستزيد المخزون)
- إنشاء فواتير بيع (ستخصم من المخزون)

### 3. اختبار المخزون
1. اذهب للمخزون وشاهد الكميات الحالية
2. أنشئ فاتورة شراء وأضف منتج
3. ارجع للمخزون → ستجد الكمية زادت ✅
4. أنشئ فاتورة بيع وأضف نفس المنتج
5. ارجع للمخزون → ستجد الكمية نقصت ✅

### 4. مراجعة الإحصائيات
اذهب للوحة التحكم وشاهد:
- إجمالي المبيعات
- إجمالي المشتريات
- المصروفات
- صافي الربح (المبيعات - المشتريات - المصروفات)

## 🔄 سير العمل النموذجي

### سيناريو كامل:

1. **إضافة مورد جديد**
   - اذهب لـ `/purchases/suppliers`
   - اضغط "إضافة مورد"
   - املأ البيانات واحفظ

2. **إضافة منتج (خامة)**
   - اذهب لـ `/inventory`
   - اضغط "إضافة منتج"
   - اختر النوع: خامة
   - احفظ (المخزون = 0)

3. **شراء الخامة**
   - اذهب لـ `/purchases/invoices`
   - اضغط "إضافة فاتورة"
   - اختر المورد
   - أضف المنتج (الخامة) بكمية 100
   - احفظ
   - **النتيجة:** المخزون أصبح 100 ✅

4. **إضافة عميل**
   - اذهب لـ `/sales/customers`
   - اضغط "إضافة عميل"
   - املأ البيانات واحفظ

5. **بيع منتج**
   - اذهب لـ `/sales/invoices`
   - اضغط "إضافة فاتورة"
   - اختر العميل
   - أضف منتج بكمية 20
   - احفظ
   - **النتيجة:** المخزون نقص 20 ✅

6. **مراجعة النتائج**
   - اذهب لـ `/` (لوحة التحكم)
   - شاهد الإحصائيات المحدثة
   - شاهد آخر الفواتير
