# دليل التثبيت والتشغيل

## خطوات التثبيت

### 1. تثبيت الحزم
```bash
npm install
```

### 2. إعداد قاعدة البيانات
```bash
npx prisma generate
npx prisma db push
```

### 3. (اختياري) إضافة بيانات تجريبية
يمكنك إضافة بيانات تجريبية من خلال واجهة النظام بعد التشغيل.

### 4. تشغيل المشروع
```bash
npm run dev
```

### 5. فتح المتصفح
افتح المتصفح على العنوان:
```
http://localhost:3000
```

## الهيكل الأساسي للنظام

### المشتريات
- **الموردين**: إضافة وإدارة الموردين
- **فواتير الشراء**: إنشاء فواتير شراء (تحديث تلقائي للمخزون)
- **أوامر الشراء**: إدارة أوامر الشراء
- **المصروفات**: تسجيل المصروفات

### المبيعات
- **العملاء**: إضافة وإدارة العملاء
- **فواتير البيع**: إنشاء فواتير بيع (خصم تلقائي من المخزون)
- **أوامر البيع**: إدارة أوامر البيع

### المخزون
- **المنتجات**: إدارة الخامات والمنتجات النهائية
- **تتبع المخزون**: تحديث تلقائي عند كل عملية
- **تنبيهات**: تنبيهات عند الوصول للحد الأدنى

### لوحة التحكم
- إحصائيات المبيعات والمشتريات
- صافي الربح
- أحدث العمليات
- تنبيهات المخزون

## البدء السريع

1. أضف موردين من قسم "المشتريات > الموردين"
2. أضف عملاء من قسم "المبيعات > العملاء"
3. أضف منتجات من قسم "المخزون"
4. أنشئ فاتورة شراء لزيادة المخزون
5. أنشئ فاتورة بيع (سيتم خصم المخزون تلقائياً)
6. راجع لوحة التحكم للإحصائيات

## الدعم الفني

النظام جاهز للاستخدام الفوري في بيئة الإنتاج.
جميع العمليات تعمل بشكل كامل مع قاعدة بيانات SQLite.

للترقية لقاعدة بيانات أكبر (PostgreSQL, MySQL)، قم بتعديل ملف `prisma/schema.prisma`.
