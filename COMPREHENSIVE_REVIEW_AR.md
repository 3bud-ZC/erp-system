## 📱 كيفية استخدام النظام

### 1️⃣ تسجيل الدخول

**الخطوات:**
1. افتح المتصفح على: http://localhost:3000
2. سيتم تحويلك تلقائيًا إلى صفحة تسجيل الدخول
3. استخدم الحساب الافتراضي:
   ```
   البريد الإلكتروني: admin@example.com
   كلمة المرور: admin12345
   ```
4. اضغط "تسجيل الدخول"

**النتيجة:** ستدخل إلى لوحة التحكم الرئيسية

### 2️⃣ لوحة التحكم الرئيسية

**ما ستراه:**
- 📊 إحصائيات المبيعات
- 📦 حالة المخزون
- 💰 الأرباح والخسائر
- 📈 الرسوم البيانية
- 🔔 التنبيهات المهمة
- 📋 آخر الأنشطة

**الأقسام الجانبية:**
```
🏠 الرئيسية
📦 المخزون
   ├─ المنتجات
   ├─ المجموعات
   ├─ الوحدات
   ├─ المستودعات
   └─ الشركات
💰 المبيعات
   ├─ العملاء
   ├─ أوامر البيع
   ├─ فواتير المبيعات
   └─ التقارير
🛒 المشتريات
   ├─ الموردين
   ├─ أوامر الشراء
   ├─ فواتير المشتريات
   ├─ المصروفات
   └─ التقارير
🏭 التصنيع
   ├─ أوامر الإنتاج
   ├─ دراسة التكاليف
   └─ العمليات
📊 المحاسبة
   ├─ قيود اليومية
   └─ الأرباح والخسائر
🏢 المستودعات
```

### 3️⃣ إدارة المخزون

**الخطوات:**
1. من القائمة الجانبية، اضغط على "المخزون"
2. اختر "المنتجات"
3. يمكنك:
   - ✅ عرض جميع المنتجات
   - ➕ إضافة منتج جديد
   - ✏️ تعديل منتج موجود
   - 🗑️ حذف منتج
   - 🔍 البحث عن منتج
   - 📊 عرض التقارير

**مثال: إضافة منتج جديد**
1. اضغط زر "➕ إضافة منتج"
2. املأ البيانات:
   - اسم المنتج
   - الكود
   - السعر
   - الكمية
   - الوحدة
   - المجموعة
3. اضغط "حفظ"

### 4️⃣ إدارة المبيعات

**الخطوات:**
1. اضغط على "المبيعات" من القائمة
2. اختر "فواتير المبيعات"
3. يمكنك:
   - ✅ عرض جميع الفواتير
   - ➕ إنشاء فاتورة جديدة
   - 📄 طباعة فاتورة
   - 💰 تسجيل دفعة
   - 📊 عرض التقارير

**مثال: إنشاء فاتورة مبيعات**
1. اضغط "➕ فاتورة جديدة"
2. اختر العميل
3. أضف المنتجات:
   - اختر المنتج
   - حدد الكمية
   - السعر سيظهر تلقائيًا
4. المجموع سيُحسب تلقائيًا
5. اضغط "حفظ"

### 5️⃣ إدارة المشتريات

**الخطوات:**
1. اضغط على "المشتريات"
2. اختر "فواتير المشتريات"
3. نفس خطوات المبيعات لكن مع الموردين

### 6️⃣ التصنيع

**الخطوات:**
1. اضغط على "التصنيع"
2. اختر "أوامر الإنتاج"
3. يمكنك:
   - إنشاء أمر إنتاج جديد
   - تحديد المواد الخام المطلوبة
   - حساب التكاليف
   - تتبع حالة الإنتاج

### 7️⃣ المحاسبة

**الخطوات:**
1. اضغط على "المحاسبة"
2. اختر "الأرباح والخسائر"
3. ستجد:
   - 💰 إجمالي الإيرادات
   - 💸 إجمالي المصروفات
   - 📊 صافي الربح/الخسارة
   - 📈 الرسوم البيانية
