# نظام ERP - مصنع البلاستيك

نظام إدارة موارد المؤسسة الكامل باللغة العربية لمصنع البلاستيك

## المميزات

### المشتريات
- أوامر شراء
- فواتير شراء (مع تحديث تلقائي للمخزون)
- إدارة الموردين
- المصروفات
- تقارير المشتريات

### المبيعات
- فواتير بيع (مع تحديث تلقائي للمخزون)
- أوامر بيع
- إدارة العملاء
- تقارير المبيعات

### التصنيع
- نظام التصنيع
- أوامر إنتاج (مع خصم خامات وإضافة منتج نهائي)
- الإنتاج التفصيلي
- عمليات الإنتاج
- دراسة التكاليف

### المخزون
- تتبع المخزون التلقائي
- تنبيهات الحد الأدنى
- خامات ومنتجات

### المحاسبة
- تتبع الإيرادات
- تتبع المصروفات
- حساب صافي الربح

## التثبيت

1. تثبيت الحزم:
```bash
npm install
```

2. إعداد قاعدة البيانات:
```bash
npx prisma generate
npx prisma db push
```

3. تشغيل المشروع:
```bash
npm run dev
```

4. فتح المتصفح على:
```
http://localhost:3000
```

## التقنيات المستخدمة

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Prisma (SQLite)
- Lucide React Icons
- Cairo & Tajawal Fonts

## الهيكل

```
app/
├── (dashboard)/          # صفحات النظام
│   ├── page.tsx         # لوحة التحكم

│   ├── sales/           # المبيعات
│   ├── manufacturing/   # التصنيع
│   └── inventory/       # المخزون
├── api/                 # API Routes
└── globals.css          # الأنماط العامة

components/              # المكونات القابلة لإعادة الاستخدام
├── Sidebar.tsx
├── Topbar.tsx
├── Table.tsx
├── Modal.tsx
└── Card.tsx

prisma/
└── schema.prisma        # مخطط قاعدة البيانات
```
