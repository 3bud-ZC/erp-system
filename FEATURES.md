## 🎯 المنطق التجاري المنفذ

### عند إنشاء فاتورة شراء:
1. حفظ الفاتورة في قاعدة البيانات
2. حفظ عناصر الفاتورة
3. **زيادة المخزون** لكل منتج بالكمية المشتراة
4. استخدام Transaction لضمان تنفيذ كل الخطوات أو عدم تنفيذ أي منها

### عند إنشاء فاتورة بيع:
1. حفظ الفاتورة في قاعدة البيانات
2. حفظ عناصر الفاتورة
3. **خصم المخزون** لكل منتج بالكمية المباعة
4. استخدام Transaction لضمان تنفيذ كل الخطوات أو عدم تنفيذ أي منها

### عند تعديل فاتورة:
1. إرجاع المخزون للحالة السابقة
2. حذف العناصر القديمة
3. إضافة العناصر الجديدة
4. تحديث المخزون حسب العناصر الجديدة

### عند حذف فاتورة:
1. إرجاع المخزون (عكس العملية الأصلية)
2. حذف الفاتورة وعناصرها
