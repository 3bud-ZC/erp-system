## 📊 الأقسام الرئيسية

### 🏠 الصفحة الرئيسية (Dashboard)
- عرض الإحصائيات العامة
- رسوم بيانية للمبيعات
- حالة المخزون
- آخر الأنشطة

### 📦 المخزون
**المنتجات:**
- عرض جميع المنتجات
- إضافة منتج جديد (زر ➕)
- تعديل منتج (زر ✏️)
- حذف منتج (زر 🗑️)

**المجموعات:**
- تصنيف المنتجات
- إضافة مجموعة جديدة

**الوحدات:**
- كيلو، قطعة، صندوق، إلخ
- إضافة وحدة جديدة

**المستودعات:**
- إدارة مستودعات متعددة
- نقل المخزون بين المستودعات

### 💰 المبيعات

**العملاء:**
1. اضغط "المبيعات" → "العملاء"
2. اضغط "➕ عميل جديد"
3. املأ البيانات (الاسم، الهاتف، العنوان)
4. احفظ

**فواتير المبيعات:**
1. اضغط "المبيعات" → "فواتير المبيعات"
2. اضغط "➕ فاتورة جديدة"
3. اختر العميل
4. أضف المنتجات:
   - اختر المنتج
   - حدد الكمية
   - السعر يظهر تلقائيًا
5. المجموع يُحسب تلقائيًا
6. احفظ

### 🛒 المشتريات

**الموردين:**
- نفس خطوات العملاء

**فواتير المشتريات:**
- نفس خطوات فواتير المبيعات لكن مع الموردين

**المصروفات:**
- تسجيل المصروفات اليومية
- تصنيف المصروفات

### 🏭 التصنيع

**أوامر الإنتاج:**
1. اضغط "التصنيع" → "أوامر الإنتاج"
2. اضغط "➕ أمر إنتاج جديد"
3. اختر المنتج النهائي
4. حدد الكمية المطلوبة
5. أضف المواد الخام المطلوبة
6. النظام يحسب التكاليف تلقائيًا

**دراسة التكاليف:**
- عرض تكاليف الإنتاج
- تحليل الأرباح

### 📊 المحاسبة

**قيود اليومية:**
- تُنشأ تلقائيًا مع كل عملية
- يمكن إضافة قيود يدوية

**الأرباح والخسائر:**
- عرض الإيرادات
- عرض المصروفات
- حساب صافي الربح

## 📊 التقارير

### تقارير المبيعات
1. اضغط "المبيعات" → "التقارير"
2. اختر الفترة الزمنية
3. شاهد:
   - إجمالي المبيعات
   - أفضل المنتجات
   - أفضل العملاء

### تقارير المشتريات
- نفس خطوات تقارير المبيعات

### تقارير المخزون
1. اضغط "المخزون" → "المنتجات"
2. زر "📊 تقرير المخزون"
3. شاهد:
   - الكميات المتاحة
   - المنتجات الناقصة
   - قيمة المخزون

## 🎯 سيناريو عملي كامل

### مثال: بيع منتج لعميل

**الخطوة 1: تأكد من وجود المنتج**
```
المخزون → المنتجات → تحقق من الكمية
```

**الخطوة 2: تأكد من وجود العميل**
```
المبيعات → العملاء → إذا لم يكن موجود، أضفه
```

**الخطوة 3: أنشئ فاتورة المبيعات**
```
المبيعات → فواتير المبيعات → ➕ فاتورة جديدة
1. اختر العميل
2. أضف المنتج والكمية
3. احفظ
```

**النتيجة:**
- ✅ الفاتورة محفوظة
- ✅ المخزون تم تحديثه (نقص الكمية)
- ✅ قيد محاسبي تم إنشاؤه تلقائيًا
- ✅ حساب العميل تم تحديثه

## 🔐 الأمان

### تغيير كلمة المرور

### إضافة مستخدم جديد
1. الإعدادات → المستخدمين
2. ➕ مستخدم جديد
3. حدد الصلاحيات
